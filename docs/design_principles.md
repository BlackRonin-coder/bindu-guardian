# Design Principles

Bindu Guardian is guided by the following principles:

- people first
- continuity where safely possible
- proportionate response
- selective containment
- human override
- auditable reasoning
- no uncontrolled autonomy

## Interpretation

These principles define behavioural boundaries for the architecture.

They ensure that defensive actions:
- minimise unnecessary disruption
- preserve essential function
- remain understandable to human operators
