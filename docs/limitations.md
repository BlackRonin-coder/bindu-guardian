# Limitations

## Public release limitations

This repository does not include:

- production-ready deployment code
- internal optimisation logic
- full system integrations
- operational tuning parameters

## Architectural limitations

Bindu Guardian is not a replacement for human authority.

It is designed to:
- support decision-making
- improve signal interpretation
- enhance clarity and proportionality

## Principle

All outputs should remain:
- governed
- explainable
- reviewable
