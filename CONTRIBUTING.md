# Contributing

Thank you for your interest in Bindu Guardian.

## Public Contribution Boundary

This repository is a protected public release. Contributions should improve clarity, structure, presentation, and public-facing documentation without exposing protected internal logic.

## Appropriate contributions

- documentation improvements
- wording clarity
- architectural communication
- typo fixes
- safe demo improvements

## Not appropriate for public contribution

- requests to expose protected decision logic
- requests for sensitive thresholds or tuning
- changes that weaken governance or public-safety framing
- additions that imply uncontrolled autonomy

## Principle

Bindu Guardian should remain:

- governed
- explainable
- safety-conscious
- continuity-preserving
