# Bindu Guardian

Bindu Guardian is a governed decision architecture for high-risk systems.

## What it does

It is designed to:
- detect system instability
- resist manipulation
- coordinate multi-agent inputs
- classify real-world scenarios
- provide explainable decisions

## Key Concept

Most systems react.

Bindu Guardian:
- understands context
- tracks change over time
- classifies the situation
- responds proportionally
- explains why
