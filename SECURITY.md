# Security Policy

## Public Release Boundary

This repository is a protected public release of Bindu Guardian.

It is intended to communicate architecture, design principles, and example behaviour without exposing sensitive implementation details.

## Not Included

The public repository does not include:

- protected internal decision logic
- production thresholds
- tuning methods
- sensitive deployment details
- full operational integrations

## Reporting

If you identify a security issue in the public repository, report it privately through the contact route listed in `docs/company.md`.

Do not publish suspected vulnerabilities publicly before responsible review.
